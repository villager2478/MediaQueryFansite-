What Is Fortnite?

Fortnite is an online battle royale and creative building game where 100 players drop onto an island and fight to be the last squad standing. Players gather weapons, build structures, and rotate with the storm to survive.

The game constantly changes with new seasons, maps, crossovers, and live events that feel like interactive art pieces.

Core Game Modes

  • Battle Royale: 100 player last person standing mode.
  • Zero Build: No building, pure aim and positioning.
  • Creative: Community made maps, deathruns, and minigames.
  • Ranked: Competitive mode with skill-based tiers.
